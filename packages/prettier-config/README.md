# Prettier Config of taoismcn

## Install

```shell
# with npm
npm i @taoismcn/prettier-config -D

# with yarn
yarn add @taoismcn/prettier-config -D

# with pnpm
pnpm add @taoismcn/prettier-config -D
```

## Usage

```js
// .prettierrc.js

const config = require('@taoismcn/prettier-config')

module.exports = {
	...config
}
```
